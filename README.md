# Tauben-Website (lokal)

Diese kleine Webseite liefert kompakte und fundierte Informationen über Tauben und enthält eine Bildergalerie.

Dateistruktur:
- index.html — Hauptseite
- styles.css — Styling
- script.js — Lightbox-Script für die Galerie
- images/ — Ordner für Bilder (nicht enthalten)

Anleitung:
1. Lege einen Ordner an und speichere die oben genannten Dateien darin.
2. Lege im Ordner einen Unterordner `images/` an und füge dort diese Dateien ein:
   - pigeon1.jpg
   - pigeon2.jpg
   - pigeon3.jpg
   (Dateinamen müssen mit denen in `index.html` übereinstimmen oder passe die Pfade an.)
3. Öffne `index.html` im Browser. Alternativ kannst du in VS Code die Erweiterung "Live Server" nutzen, um die Seite lokal zu hosten.

Empfohlene Bildquellen (frei nutzbar / Commons/Unsplash — jeweils Lizenz prüfen):
